# SectorV

MD Hack project.

Hosted at [go/sector-v](https://sector-v-a531a.web.app/) or at https://sector-v-a531a.web.app/.

## Quickstart

```bash
npm i -g polymer-cli
npm i -g firebase-tools
firebase login

# dev local
npm i && npm run watch & npm run dev

# preview prod
npm i && npm run build:prod && npm run dev:prod

# deploy
npm i && npm run deploy
```
